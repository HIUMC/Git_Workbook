# 1주차 학습 내용 정리

##  Git 기초
- `git init` : 새로운 Git 저장소 생성
- `git clone <repo>` : 원격 저장소 복제
- `git add <파일>` : 변경된 파일을 스테이징
- `git commit -m "메시지"` : 변경 사항 저장
- `git push origin main` : 원격 저장소에 푸시

##  브랜치 관리
- `git branch` : 현재 브랜치 확인
- `git checkout -b <브랜치>` : 새 브랜치 생성 후 이동
- `git merge <브랜치>` : 브랜치 병합
